# SunBuggy Fun Rentals Booking App

Welcome to the SunBuggy Fun Rentals Booking App! This app allows you to easily book exciting outdoor activities and rentals in Las Vegas, Nevada. Whether you're looking for off-road adventures, dune buggy rides, or ATV rentals, we've got you covered.

## Features

- Browse and search for a wide range of outdoor activities and rentals.
- View detailed descriptions, photos, and pricing information for each activity.
- Check availability and book your preferred activity directly from the app.
- Customize your booking by selecting the date, time, and number of participants.
- Receive instant confirmation and booking details via email.
- Manage your bookings, view past reservations, and make changes if needed.

## Installation

To use the SunBuggy Fun Rentals Booking App, follow these steps:

1. Clone this repository to your local machine.
2. Install the required dependencies by running `npm install`.
3. Start the app by running `npm start`.
4. Open your web browser and navigate to `http://localhost:3000`.

## Usage

Once the app is up and running, you can start exploring the available activities and rentals. Use the search bar to find specific activities or browse through the categories. Click on an activity to view more details and make a booking.
